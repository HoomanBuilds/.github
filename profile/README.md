### What we do

Hackathon prototypes that become real products. We operate at the intersection of Web3, AI, and full-stack development — building fast under pressure, then shipping to production. Every project in our repos started as a 24-72 hour build and graduated to a live product.

### Featured Projects

**[Codrel](https://github.com/hoomanbuilds/codrel):**
Context engine for AI coding agents. Ingests project knowledge into RAG-ready format and serves it to IDEs via MCP, eliminating hallucination.
Winner, AWS Vibe Coding Hackathon — $10K AWS Credits + $38K+ in prizes

**[Chartor Bot](https://github.com/HoomanBuilds/chartor-bot):**
AI-powered algorithmic trading bot using GPT-5.1 decision-making with multi-timeframe technical analysis for WEEX futures.
Finalist, AI Wars: WEEX Alpha Awakens — $880K prize pool

**[Flash Protocol](https://github.com/hoomanbuilds/flash-protocol):**
Non-custodial cross-chain payment gateway. Merchants accept any token across 70+ networks with automatic USDC settlement via 6 liquidity providers.

**[OpenScene](https://github.com/HoomanBuilds/OpenScenes):**
AI-powered video presentation engine. Multi-agent pipeline with Gemini, tiered natural language editing, distributed render engine with Remotion.

**[Signatory](https://github.com/HoomanBuilds/Signatory):**
Protocol for autonomous AI agents with cryptographic signing authority. Lit Protocol MPC wallets, GOAT SDK DeFi execution and x402 monetization.
